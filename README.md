###Exercício 1 – Analisador de Lista de Compras

Crie um programa que:
	1.	Crie uma lista contendo alguns alimentos (por exemplo: frutas ou itens de mercado).
	2.	Percorra essa lista utilizando um for.
	3.	Exiba cada item da lista individualmente na tela.
	4.	Pergunte ao usuário um alimento.
	5.	Verifique se esse alimento está presente na lista.

Caso esteja na lista, informe que o item já está na lista de compras.
Caso não esteja, informe que o item não foi encontrado.

Dicas sutis:
	•	Lembre-se de que listas são sequências que podem ser percorridas com for.
	•	Existe um operador específico para verificar se um elemento está dentro de uma sequência.
	•	Primeiro percorra a lista, depois faça a verificação.

###Exercício 2 – Contador de Vogais

Crie um programa que:
	1.	Peça ao usuário para digitar uma palavra.
	2.	Percorra cada letra dessa palavra utilizando um for.
	3.	Conte quantas vogais existem na palavra.
	4.	Ao final, exiba a quantidade total de vogais encontradas.

Considere as seguintes vogais:
a, e, i, o, u

Dicas sutis:
	•	Strings também podem ser percorridas com for.
	•	Use uma variável para armazenar a quantidade de vogais encontradas.
	•	A cada vogal identificada, aumente esse contador.

⸻

###Exercício 3 – Sistema de Senha com Tentativas

Crie um programa que:
	1.	Defina uma senha correta dentro do código.
	2.	Peça para o usuário digitar uma senha.
	3.	Continue pedindo a senha enquanto ela estiver incorreta.
	4.	Conte quantas tentativas o usuário fez.
	5.	Quando ele acertar, exiba uma mensagem informando que o acesso foi liberado e quantas tentativas foram necessárias.

Dicas sutis:
	•	Utilize um laço while.
	•	Crie uma variável para armazenar o número de tentativas.
	•	Lembre-se de atualizar o valor digitado dentro do laço.
